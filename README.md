🍳 RecipeRAG AI – Intelligent Document Q&A Recipe Generator

Transform recipe documents into an intelligent AI cooking assistant using Retrieval-Augmented Generation (RAG), IBM Langflow, IBM Orchestrate, and IBM Granite Foundation Models.

RecipeRAG AI is an AI-powered Retrieval-Augmented Generation (RAG) application that converts unstructured recipe documents into an interactive cooking assistant. Users can upload cookbooks, PDFs, food blogs, or personal recipe collections and ask natural language questions to receive accurate, context-aware, and personalized cooking guidance.

Powered by IBM Langflow, IBM Orchestrate, and IBM Granite Models, the platform retrieves relevant recipes from a semantic knowledge base and adapts them based on dietary restrictions, allergies, available ingredients, cuisine preferences, and cooking time. Beyond recipe retrieval, it also provides nutrition insights, ingredient substitutions, shopping lists, and step-by-step cooking instructions.

🚀 How It Works
Upload Recipe Documents
Import recipe PDFs, cookbooks, food blogs, or handwritten recipe notes.
Document Processing
Documents are parsed, chunked, embedded, and stored in a vector database.
Semantic Retrieval
The RAG pipeline retrieves the most relevant recipe information based on the user's query.
AI Reasoning
IBM Granite Models analyze the retrieved context to generate accurate, context-aware responses.
Personalized Cooking Assistance
The system adapts recipes based on ingredients, dietary restrictions, allergies, cooking time, and cuisine preferences.
