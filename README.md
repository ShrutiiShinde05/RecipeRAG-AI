# 🍳 RecipeRAG AI – Intelligent RAG-Powered Recipe Assistant

> **Transform recipe documents into an intelligent AI cooking assistant using Retrieval-Augmented Generation (RAG), IBM Langflow, IBM Orchestrate, and IBM Granite Foundation Models.**

---

## 📖 Overview

RecipeRAG AI is an intelligent **Retrieval-Augmented Generation (RAG)** application that transforms unstructured recipe documents into an interactive AI cooking assistant.

Using **IBM Langflow**, **IBM Orchestrate**, and **IBM Granite Foundation Models**, the platform enables users to upload cookbooks, PDFs, food blogs, and personal recipe notes, then ask natural language questions to receive accurate, context-aware, and personalized cooking guidance.

The system retrieves relevant recipes from a semantic knowledge base and adapts them based on:

- 🥗 Dietary preferences
- 🥜 Food allergies
- 🛒 Available ingredients
- ⏱️ Cooking time
- 🌎 Cuisine preferences

It also provides ingredient substitutions, nutrition analysis, shopping lists, and step-by-step cooking instructions—all powered by **Agentic AI**.

# 🚀 How It Works

1. Upload recipe documents.
2. Extract and preprocess recipe content.
3. Generate embeddings and store them in a vector database.
4. Retrieve relevant recipes using RAG.
5. IBM Granite Models generate context-aware responses.
6. IBM Orchestrate coordinates AI agents for personalized cooking assistance.
